# Compiling

1. Qt5 must be installed.
2. Set ``QtPath`` in "CMakeLists.txt" if on Windows.
3. Run ``CMake``.

# Recommended IDE

1. Visual studio code.
2. [CMake Tools extension](https://marketplace.visualstudio.com/items?itemName=vector-of-bool.cmake-tools).
